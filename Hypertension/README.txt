# Hypertension Model

## Description
This project is a predictive model that aims to determine whether a person is at risk of developing hypertension in the future.  
The model uses **classification methodology**, specifically the **Random Forest** algorithm.

## Feature Engineering
In this model, we applied several **feature engineering techniques**, including:  
- Encoding textual features  
- Removing duplicate records  
- Filling empty cells  
- Applying cross-validation for better model evaluation  

## How to Run

python app.py
